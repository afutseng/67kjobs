關於我們


所以你是說你的薪水不夠高。


你有考慮過到國外工作嗎？不不不，人不用到國外。比方說 JOBMOTE 這個網站提供了不少遠距工作的職缺，像是 這一個職缺 所需的技能差不多就只是架設這一個平台所需的技能而已。別忘了也參考一下Flexjobs 這一個網站，看看他的標題：

Life is flexible... is your job?

我想這個問題必須要由你自己來回答。


這個平台的建立是啟發自66kjobs，據說開發者使用 Ruby 花了八個小時就把網站建立起來。我想這會是一個很好的練習題目，稍微思考了一下，決定為了節省時間，連後端都不開發了，純粹撰寫前端程式碼，使用簡潔的 jade 、 sass 、 livescript 、 bootstrap 、 angularjs 、 firebase 稍微兜兜兜，咦！才兩個多小時，我已經開始在寫「關於我們」了，其中一個小時還是在爬取職業列表以及挑選首頁用的照片呢，就只差 inside jobs 爬蟲還沒寫了，不過我傾向去爬批踢踢的 job / tech-job / soft-job / part-time 各板，反正 批踢踢爬蟲程式 以前就已經寫過了，只差文章 parser 而已。


67kjobs 後端接的是 firebase ，權限控制上還有點問題，有機會可以嘗試用 python django 或是 nodejs express + mongodb 再寫一次，相信時間應該同樣不會花超過三個小時。


所有程式碼皆放在 github repo ，歡迎自己拿去參考。
